# Customer-Management-System-DSA
A C++ Customer Management System implementing core Data Structures and Algorithms from scratch, including Linked Lists, Stack, Queue, BST, AVL Tree, Min Heap, Hash Table, Graph Algorithms, and Sorting Techniques.

A complete **Customer Management System** built in **C++** that demonstrates the practical implementation of fundamental **Data Structures and Algorithms (DSA)** without relying on STL data structures for the core implementations.

The project manages **1000 customer records** loaded from a CSV dataset and integrates multiple data structures into one real-world business application.

---

## Overview

This project was developed as the semester final project for the **Data Structures** course at **COMSATS University Islamabad**.

Unlike standalone implementations of data structures, this project combines multiple structures into one cohesive customer management system where every data structure solves a specific business problem.

The system loads customer information from a CSV dataset and provides operations for searching, sorting, indexing, customer management, undo functionality, market analysis, and graph-based route optimization.

## Features

###  Customer Management

- Load customer records from CSV
- Display all customers
- Reverse traversal
- Search customers by name
- Count customers by country
- Delete customers by country

---

### Undo System (Stack)

- Store deleted customer records
- Undo last deletion
- Display undo history

---

### Processing Queue

- Enqueue customers
- Dequeue customers
- Display processing queue

---

### Binary Search Tree

- Build BST
- Search customer by ID
- Delete customer
- Inorder traversal

---

### AVL Tree

- Self-balancing customer index
- O(log n) searching
- Automatic tree rotations

---

### Min Heap

- Priority queue based on subscription date
- Retrieve oldest subscriber
- Display priority queue

---

### Hash Table

- O(1) average lookup
- Search customer by ID
- Delete customer
- Display hash table

---

### Sorting Algorithms

- Bubble Sort (Last Name)
- Selection Sort (Subscription Date)
- Merge Sort (Country)
- Binary Search (After Sorting)

---

### Graph Algorithms

- Depth First Search (DFS)
- Breadth First Search (BFS)
- Dijkstra Algorithm
- Prim's MST
- Kruskal's MST
